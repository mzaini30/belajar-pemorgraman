Menggabungkan File PDF

Install dulu pdftk:

sudo apt install pdftk

Misal, kita ingin menggabungkan a.pdf dengan b.pdf menjadi c.pdf, maka perintahnya adalah:

pdftk a.pdf b.pdf cat output c.pdf
