Olah Array di JavaScript

Misal aja ada array:

let angka = [1, 2, 3, 4, 5]

Tambah

Misal, ditambahkan angka 6:

angka = [...angka, 6]

Hapus

Hapus angka 4:

angka = angka.filter(x => x != 4)

Update

Update angka 3 menjadi 'tiga':

angka = angka.map(x => x == 3 ? 'tiga' : x)
